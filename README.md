# adventures-work-dashboards
Project:

Developed a data analysis project using sample raw data from AdventureWorks, a global manufacturer of cycling equipment and accessories, to gain valuable insights that helped track KPIs (Sales, Revenue, Profit, Returns), compare regional performance, analyze product trends, and identify high-value customers.

#1 Load and prepare data into Power BI Desktop

I Transformed tables using text, numerical, and date/time tools, pivoted and grouped records, and created new conditional columns. Reviewed tools for key profiling metrics such as column distribution, empty values, errors, etc.

#2 Created a data model

Performed normalization, fact and dimension tables, primary and foreign key relationships, cardinality types, report filter configurations, and data modeling options like hierarchies and hidden fields.

#3 Created a dashboard to visualize and analyse data

Practised on DAX for adding calculated columns and measures, understanding row and filter context, DAX formula syntax, operators, and common functions, as well as inserting, formatting, and filtering visuals in report view and adding interactivity with bookmarks, slicers, and tooltips,

Here's to continuous learning and growth!
